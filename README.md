# Quantium Retail Sales Analysis

## Project Overview

This project analyzes retail transaction data from Quantium to understand customer purchasing behavior, product performance, and sales trends. The analysis focuses on identifying actionable business insights through data cleaning, feature engineering, exploratory data analysis (EDA), and customer segmentation.

---

## Objectives

- Clean and prepare the retail transaction dataset.
- Engineer new features such as brand names and pack sizes.
- Analyze customer demographics and purchasing behavior.
- Identify the highest-performing brands and products.
- Examine sales trends over time.
- Generate business insights and recommendations.

---

## Dataset

The analysis uses two datasets:

- Transaction dataset
- Customer dataset

These datasets were merged using the `LYLTY_CARD_NBR` field.

## Key Insights

- Kettle was the highest-performing brand by total sales.
- Medium-sized pack sizes (150g–175g) generated the highest revenue.
- Older Singles/Couples and Retirees contributed significantly to total sales.
- Mainstream customers generated the highest total revenue.
- Monthly sales remained relatively stable throughout the analysis period.

---

## Business Recommendations

- Prioritize inventory for high-performing brands.
- Maintain stock levels for popular pack sizes.
- Target marketing campaigns toward high-value customer segments.
- Continue monitoring monthly sales trends.
